# Support

FilePilot AI is an open-source project. The fastest way to get help is to open a
focused GitHub issue with enough context for another contributor to reproduce
the problem.

## Before Opening An Issue

- Check [README.md](README.md) for quick start instructions.
- Check [docs/MCP.md](docs/MCP.md) for MCP setup and troubleshooting.
- Check [docs/MCP-CLIENTS.md](docs/MCP-CLIENTS.md) for client configuration.
- Check [docs/MCP-WORKFLOWS.md](docs/MCP-WORKFLOWS.md) for safe agent workflow prompts.

## What To Include

- FilePilot version or commit SHA.
- Operating system and Python version.
- Install method, such as source checkout, pip extra, or packaged app.
- The exact command or UI workflow that failed.
- Logs or screenshots, with private file contents redacted.

## Privacy

Do not paste private documents, API keys, personal data, or sensitive local paths
into public issues. For security-sensitive behavior, follow [SECURITY.md](SECURITY.md).
